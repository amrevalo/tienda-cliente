Bienvenido al proyecto Tienda

El proyecto de Alberto que gestiona una tienda fisica

¿Qué es este proyecto?

  • Este proyecto se ha hecho con el fin de aprender programacion y saber estructurar bien el codigo ademas de aprobar las asignaturas.
  • Este proyecto tiene como finalidad gestionar una tienda fisica.
  • La estructura del proyecto parte de GESANE tanto cliente como servidor.

¿Que impulsa este proyecto?

  • El autor de la aplicación base es Alberto Marco Arévalo parte del cogido es de un pryecto de clase llamado Gesane.
  • La aplicación ha sido desarrollada con el fin de exponerla a la junta de profesores y poder aprobar el curso.

¿Qué pretendemos con este proyecto?

  • Modelizar procesos de un mantenimiento de tienda y al mismo tiempo aprender a programar y a buscarse los recursos por uno mismo.

Para mi que soy un alumno de Desarrollo de aplicaciones web...

  • Aprender a estructurar el diseño de un proyecto web
  • Simular la creación de un proyecto complejo en un entorno de trabajo
  • Participar de manera práctica en la creación de una aplicación web compleja
